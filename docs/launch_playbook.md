# Launch Playbook: Referral Giveaway

## 1) Channel Announcement Post

Розыгрыш для подписчиков!

1. Подпишитесь на канал.
2. Нажмите `/start` у бота.
3. Пройдите обязательный опрос в боте (возраст и вид спорта).
4. Нажмите кнопку `Проверить подписку`.
5. Получите личную ссылку и отправьте друзьям.
6. За каждого друга, который подписался и подтвердил участие в боте, вы получаете +1 билет.

Приз: [релевантный приз для вашей аудитории].
Итоги: [дата и время].

## 2) Rules (Short)

- Участвуют и старые, и новые подписчики канала.
- Участвуют только пользователи, зарегистрированные через бота.
- Обязателен опрос (возраст и любимый спорт).
- Учитываются только подтвержденные рефералы.
- Один приглашенный пользователь учитывается один раз.
- Самоприглашения не засчитываются.
- Организатор может отклонить подозрительную активность.

## 3) Welcome Chain (after draw)

- Day 0: Пост с итогами + ссылка на лучшие материалы.
- Day 1: Короткий опрос интересов.
- Day 3: Полезный контент по теме канала.
- Day 7: Подарок для удержания (чеклист, мини-гайд, шаблон).

## 4) Admin Checklist Before Launch

- Бот добавлен в канал и имеет доступ к проверке участников.
- Активна кампания в БД (`/campaign_start`).
- Проверены deep-link и кнопка `Проверить подписку`.
- Приз и дата итогов опубликованы в посте.
- Канал имеет закреп с правилами.
