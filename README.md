# Telegram Referral Giveaway Bot

Пошаговая инструкция для запуска бота. Просто копируйте команды и вставляйте в терминал.

---

## Шаг 1. Активируйте виртуальное окружение

Скопируйте эту команду, вставьте в терминал и нажмите Enter:

```bash
.venv\Scripts\activate.bat
```

Если команда сработала, в начале строки появится `(.venv)`.

---

## Шаг 2. Проверьте подключение к базе данных

Скопируйте эту команду, вставьте в терминал и нажмите Enter:

```bash
python check_db_connection.py
```

Если всё хорошо, вы увидите сообщение `[OK] Соединение с базой данных установлено!`.

---

## Шаг 3. Проверьте подключение к Telegram

Скопируйте эту команду, вставьте в терминал и нажмите Enter:

```bash
python check_bot_connection.py
```

Если всё хорошо, вы увидите `[OK] Соединение с Telegram API успешно установлено!`.

---

## Шаг 4. Запустите бота

Скопируйте эту команду, вставьте в терминал и нажмите Enter:

```bash
python -m app.main
```

Бот запущен. Не закрывайте это окно терминала — бот работает в нём.

---

## Шаг 5. Как остановить бота

В окне терминала, где запущен бот, нажмите на клавиатуре:

```
Ctrl + C
```

Бот остановится.

---

## Шаг 6. Посмотреть список кампаний (ID)

Откройте новое окно терминала (или остановите бота), активируйте виртуальное окружение (Шаг 1) и выполните:

```bash
python list_campaigns.py
```

На экране появится таблица с кампаниями и их номерами (ID).

---

## Шаг 7. Запустить розыгрыш приза

Откройте новое окно терминала, активируйте виртуальное окружение (Шаг 1) и выполните:

```bash
python start_draw.py <ID_кампании>
```

Вместо `<ID_кампании>` подставьте номер кампании из Шага 6.

Пример:

```bash
python start_draw.py 1
```

---

## Шаг 8. Очистить базу данных (если нужно начать заново)

Откройте новое окно терминала, активируйте виртуальное окружение (Шаг 1) и выполните:

```bash
python clear_db.py
```

Скрипт спросит, что именно очистить. Следуйте подсказкам на экране.

---

## Шаг 9. Выгрузить результаты в Excel

После окончания розыгрыша откройте новое окно терминала, активируйте виртуальное окружение (Шаг 1) и выполните:

```bash
python export_to_excel.py
```

Файл Excel появится в папке `exports`.

---

## Полезные заметки

- Файл `app/texts.py` содержит тексты сообщений, которые бот отправляет пользователям.
- Файл `.env` содержит настройки: токен бота, пароль от базы данных, прокси и т.д.
- Если бот не отвечает на команды, проверьте, что он запущен (Шаг 4).
